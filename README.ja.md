# 八王子景観100 (hachioji-keikan100)

本プロジェクトは、日本の八王子市が提供するオープンデータ「八王子景観100」のデータセットを提供します。各スポットの位置情報、説明、画像を含み、整形されたCSVファイルとして提供され、インタラクティブな地図上で視覚化されています。

## デモ

インタラクティブな地図上で100の景観スポットを探索できます。

- **[標準マップビュー](https://code4fukui.github.io/hachioji-keikan100/)**: 標準的なマップピンで場所を表示します。高速で軽量です。
- **[画像アイコンマップビュー（重い）](https://code4fukui.github.io/hachioji-keikan100/useimage.html)**: 地図上に各場所のサムネイル画像を直接表示します。注意: 画像数が多いため、このバージョンは読み込みに時間がかかる場合があります。

## データ

主要なデータセットは直接利用可能です。

- **CSVファイル**: [`hachioji-keikan100.csv`](https://code4fukui.github.io/hachioji-keikan100/hachioji-keikan100.csv)

CSVファイルには以下の列が含まれます:
- `no`: 景観スポットの一意の識別子（1-100）。
- `name`: 場所の名称。
- `lat`: 緯度。
- `lon`: 経度。
- `要素` (カテゴリ): 元データのカテゴリ。
- `住所` (住所): 場所の住所。
- `説明` (説明): スポットの簡単な説明。
- `image`: スポットのメイン画像のURL。
- `nimage`: スポットで利用可能な画像の数。

## スクリプト

このリポジトリには、元データと画像を処理するためのDenoスクリプトが含まれています:

- `renames.js`: 元の画像ファイル名をクリーンアップするユーティリティ。
- `makelist.js`: 画像ファイル（例: `1-1.jpg`, `1-2.jpg`）をスキャンし、各スポットの画像総数をリストした `hachioji-keikan100-photo.csv` を生成します。
- `makecsvwithimg.js`: 元の位置情報データと画像メタデータを結合し、完全な画像URLを含む最終的な `hachioji-keikan100.csv` を生成します。
